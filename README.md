# Python Practice for Motorsports

Practicing Python fundamentals with datasets related to motorsports analytics.  
This includes data cleaning, NumPy operations, Pandas analysis, and visualization.

## What’s Inside
- `Numpy_notes.ipynb` — examples and practice with NumPy arrays
- `pandas_notes.ipynb` — exploration of data using Pandas
- `Python_ms_practice.ipynb` — practical examples with real-world motorsport lap data
- `monaco_2025_dirty_laps.csv` — sample dataset

## Key Takeaways
- Learned how to process motorsport data
- Structured notebooks for clarity

## Tools Used
Python | NumPy | Pandas
